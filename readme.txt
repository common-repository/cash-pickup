A WooCommerce extension that adds the "Cash" payment system

== Description ==

This extension adds the "Cash" payment system to your WooCommerce store.

* [WordPress Cash Pickup] (http://www.webcreates.eu/download/payment-cash-pickup.zip)

== Premium version ==

If customer chooses "local pickup", the system disables all methods of payment (bacs, checks, cod and paypal automatically) except the "cash pickups". 
If there was a system of payment other than those listed, you have to manually change that part of the code.
Buy link: http://www.webcreates.eu/sell_plugin.php

== Installation ==

= Installation Instructions =

1. Go to Plugins > Add New > Search
2. Type WooCommerce Payment Cash Pickup in the search box and hit Enter
3. Click on the button Install and then activate the plugin

= Manual Installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application.

The WordPress codex contains [instructions on how to do this here]:
http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation
